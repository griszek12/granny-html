\# 🎮 Granny Game



Granny - horror gra sieciowa hostowana na GitHub Pages!



\## 🚀 Jak to działa?



1\. Gra jest hostowana bezpłatnie na GitHub Pages

2\. Załaduje się automatycznie z CDN

3\. Brak potrzeby serwera ani instalacji



\## 📖 Instrukcja dla GitHuba



\### Dla właściciela repo:



1\. \*\*Utwórz nowe repo\*\*: `granny-game` (Public)

2\. \*\*Wrzuć pliki\*\*:

&nbsp;  - `index.html` - główny plik gry

&nbsp;  - `README.md` - ta instrukcja

3\. \*\*Włącz GitHub Pages\*\*:

&nbsp;  - Idź do: Settings → Pages

&nbsp;  - Source: main branch

&nbsp;  - Poczekaj ~1 minutę



4\. \*\*Gra będzie dostępna pod\*\*:

&nbsp;  ```

&nbsp;  https://\[TWOJ\_LOGIN].github.io/granny-game/

&nbsp;  ```



\## 🎯 Aby zagrać:



1\. Wejdź na link GitHub Pages (powyżej)

2\. Poczekaj na załadowanie gry (~5-10 sekund)

3\. Graj!



\## 📝 Cechy:



\- ✅ Pełna gra Granny

\- ✅ System reklam

\- ✅ Continue for Ads

\- ✅ Hosting za darmo

\- ✅ Brak limitów



\## ⚠️ Ważne:



\- Gra załadowuje się z CDN, musisz mieć internet

\- Pierwsza ładowanie może być wolne (gra jest duża ~50MB)

\- Używaj Chrome lub Firefox dla najlepszych wyników

\- Ad-blockery mogą blokować reklamy



\## 🐛 Jeśli coś nie działa:



1\. Otwórz F12 (Developer Tools)

2\. Sprawdź Console na błędy

3\. Przeładuj stronę (Ctrl+F5)

4\. Wyczyść cache przeglądarki
